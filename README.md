# RAG Phase 1 Project

## Setup

1. Install dependencies:
pip install -r requirements.txt

2. Add PDF files inside the data/ folder

3. Run:
python main.py

## Features
- PDF ingestion
- Chunking
- Embeddings
- FAISS vector search
- LLM-based QA
